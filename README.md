## 📋 Description

Chatbot builder platform

## 🚀 Services

- **app**: Typebot (`baptistearno/typebot-builder:latest`)

## 📦 Installation

```shell
composemgr install typebot
```

## 🔧 Configuration

```shell
TZ=America/New_York
BASE_HOST_NAME=typebot.example.com
```

- **Typebot**: http://localhost:3000

## 📂 Volumes

- Change default passwords
- Configure HTTPS with reverse proxy
- Regular backups
